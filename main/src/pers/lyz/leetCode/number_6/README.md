<h1>题目：</h1>
<p>
给定一个只包括 '('，')'，'{'，'}'，'['，']'的字符串，判断字符串是否有效。<br>
有效字符串需满足：<br>
左括号必须用相同类型的右括号闭合。<br>
左括号必须以正确的顺序闭合。<br>
注意空字符串可被认为是有效字符串。<br>
示例 1:<br>
输入: "()"<br>
输出: true<br>
示例2:<br>
输入: "()[]{}"<br>
输出: true<br>
示例3:<br>
输入: "(]"<br>
输出: false<br>
示例4:<br>
输入: "([)]"<br>
输出: false<br>
示例5:<br>
输入: "{[]}"<br>
输出: true<br>
</p>
<h1>解题思路</h1>
<p>使用栈来判断！</p>
